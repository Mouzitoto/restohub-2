# Шаг 6.1: Главная страница

## Описание
Реализация главной страницы client-web с поиском ресторанов, фильтрами, категориями и картой.

## Задачи

### 1. Поиск ресторанов
- Поисковая строка на главной странице
- Поиск по названию ресторана
- Автодополнение (опционально)
- Debounce для оптимизации запросов
- Отображение результатов поиска

### 2. Фильтры и категории
- Фильтры:
  - По типу кухни (если реализовано)
  - По рейтингу (если реализовано)
  - По наличию акций
  - По наличию тематических вечеров
  - **По залам на открытом воздухе** (чекбокс "Только с залами на открытом воздухе")
  - **По курящим залам** (чекбокс "Только с курящими залами")
- Сброс фильтров
- Комбинация нескольких фильтров
- При выборе фильтра отправляется GET запрос на `/client-api/r/search` с соответствующими query параметрами (`isOutdoor=true` или `isSmoking=true`)

### 3. Карта с ресторанами
- Интеграция карты (Google Maps, Yandex Maps, или Leaflet)
- Отображение ресторанов на карте (маркеры)
- При клике на маркер - информация о ресторане
- Переход на страницу ресторана при клике
- Центрирование карты на местоположении пользователя (опционально)
- Фильтрация ресторанов на карте по поисковому запросу

### 4. Список ресторанов
- Отображение результатов поиска/фильтров в виде карточек
- Информация в карточке:
  - Логотип ресторана
  - Название
  - Адрес
  - Краткое описание
  - Расстояние (если используется геолокация)
  - Активные акции (badge)
- Клик на карточку → переход на страницу ресторана

### 5. Пагинация
- Поддержка пагинации результатов
- Infinite scroll или кнопки "Следующая страница"
- Отображение общего количества результатов

### 6. Сортировка
- Сортировка по:
  - Расстоянию (если используется геолокация)
  - Названию
  - Популярности (если есть метрики)
  - Наличию акций

### 7. Responsive дизайн
- Адаптивность для мобильных устройств
- Оптимизация карты для мобильных
- Touch-friendly интерфейс

### 8. UX улучшения
- Loading states при загрузке результатов
- Empty states (когда нет результатов)
- Error handling (ошибки загрузки)
- Быстрые фильтры (chips/tags)
