# Шаг 4.9: Подписки

## Описание
Интерфейс для просмотра информации о подписке ресторана.

## Задачи

### 1. Страница информации о подписке
- Отображение текущего статуса подписки:
  - Тарифный план
  - Дата начала подписки
  - Дата окончания подписки
  - Статус (активна/неактивна/просрочена)
- Визуальная индикация статуса (цветные индикаторы)

### 2. Оповещения
- Предупреждение, если подписка скоро истекает (например, за 7 дней)
- Предупреждение, если подписка неактивна или просрочена

### 3. Для менеджера
- Только просмотр информации
- Невозможность изменения статуса
- Контактная информация для продления подписки

### 4. Для администратора
- Полный доступ к информации о подписке
- Возможность обновления статуса вручную
- Возможность продления подписки
- Управление тарифными планами

### 5. Дизайн
- Четкое отображение ключевой информации
- Инструкции по продлению подписки
- Контактная информация для связи (оплата происходит вне системы)
